# prompt-studio-app
Prompt Studio ✨
A modern web application for crafting, testing, and managing your AI prompts with ease. This tool provides a simple, intuitive interface to work with dynamic variables, test against Google's Gemini API, and save your best work to a personal library.

➡️ View the Live Demo (Note: Remember to replace this with your actual Netlify URL after you deploy!)

Key Features
✍️ Dynamic Variable Editor: Automatically detects {{variables}} in your prompt and creates input fields for them.

🤖 Live AI Testing: Send your compiled prompt directly to the Gemini API and see the response in real-time.

📚 Personal Prompt Library: Save, load, and manage your prompts using a secure, personal Firestore database.

🚀 Clean & Responsive UI: Built with Tailwind CSS for a great experience on any device, from mobile to desktop.

🔐 Secure & Anonymous: Uses Firebase Anonymous Authentication to save your work without requiring you to create an account.

Tech Stack
Frontend: React

Backend & Database: Firebase (Authentication & Firestore)

AI Model: Google Gemini API

Styling: Tailwind CSS

Icons: Lucide React
